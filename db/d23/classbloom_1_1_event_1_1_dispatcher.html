Detailed Description

A utility class for dispatching events to appropriate handlers.

The EventDispatcher class provides a mechanism for safely and efficiently handling events by invoking user-defined callback functions. It ensures that events are only handled by functions designed for their specific type.

bool bloom::Event::Dispatcher::Dispatch ( EventFn< T > func)
inline

Dispatches the event to a handler function if the types match.

This function compares the type of the event with the static type of the handler's event type. If they match, the handler function is called, and the event is marked as handled.
